# Student Task & Expense Manager

A beginner-friendly Python project that helps students manage school tasks and track expenses.

## Features

- Add assignments
- View assignments
- Mark assignments as completed
- Add daily expenses
- View total expenses
- Filter expenses by category
- Show expenses above a certain amount

## Concepts Used

- Lists
- Tuples
- While loops
- Conditional statements
- List comprehensions

## How to Run

```bash
python main.py
```

## Example Menu

1. Add Assignment  
2. View Assignments  
3. Mark Assignment Completed  
4. Add Expense  
5. View Expenses  
6. Filter Expenses by Category  
7. Expenses Above Amount  
0. Exit
